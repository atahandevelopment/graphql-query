#GraphQL Query Task


Önceliklle Terminalinizi açıp - git clone https://github.com/atahandevelopment/graphql-query - komutunu çalıştırınız.

Proje dosya dizininize indirildikten sonra 

1. cd client
  - npm install 

  ve

2. cd server
  - npm install  komutlarını çalıştırarak proje modüllerini yükleyiniz.

Proje yüklendikten sonra 

  1. cd server
    - npm start komutlarını çalıştırarak backend ayağa kaldırılmalıdır.

  2. cd client
    - npm run dev komutlarını çalıştırarak frontend'i ayağa kaldırabilir ve projeyi çalıştırabilirsiniz...
